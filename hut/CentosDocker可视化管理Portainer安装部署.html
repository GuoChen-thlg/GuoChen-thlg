dockerPortainer可视化管理

2021-07-31T14:47:58.000Z

Portainer 是 Docker 的图形化管理工具，提供状态显示面板、应用模板快速部署、容器镜像网络数据卷的基本操作（包括上传下载镜像，创建容器等操作）、事件日志显示、容器控制台操作、Swarm 集群和服务等集中管理和操作、登录用户管理和控制等功能。功能十分全面，基本能满足中小型单位对容器管理的全部需求

默认 Docker 环境

linux x86_64 CentOS Linux 8

Docker Version 20.10.7 (API: 1.41)

Portainer 安装与部署

cd /var/www/

mkdir portainer_data

docker run -it -d -p 9000:9000 --name portainer --restart always -v /var/run/docker.sock:/var/run/docker.sock -v /var/www/portainer_data:/data portainer/portainer-ce

访问 http://{服务器 ip}:9000
