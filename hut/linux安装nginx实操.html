linux安装nginx实操

163字小于1分钟

linuxnginx

2021-07-14T09:23:36.000Z

作为一个 😣👿 的开发，不得不在仅有的一台服务器上部署我们的项目，但多个项目如何公用一个端口，相安无事呢？答案你一定知道 😁😁

安装依赖

yum -y install gcc zlib zlib-devel pcre-devel openssl openssl-devel

下载安装包

cd /usr/local
wget http://nginx.org/download/nginx-1.20.1.tar.gz

解压缩

tar -zxvf nginx-1.20.1.tar.gz

执行配置

cd nginx-1.20.1
./configure

编译安装

make
make install

启动 nginx

/usr/local/nginx/sbin/nginx

访问你的服务器 IP 即可看到 nginx 默认页面
