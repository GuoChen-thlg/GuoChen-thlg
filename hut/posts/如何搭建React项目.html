134字小于1分钟

React

React 是什么

React 是一个声明式，高效且灵活的用于构建用户界面的 JavaScript 库。使用 React 可以将一些简短、独立的代码片段组合成复杂的 UI 界面，这些代码片段被称作“组件”。

React 项目搭建步骤

  1. 安装 Node npm 并检查其版本
node -v
npm -v
  1. 创建项目
npx create-react-app my-app
  1. 打开项目
cd my-app

npm start
